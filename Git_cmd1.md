# git commands for daily use ! 

### To check the status of the repo
`` git status ``

### To add files to the tracked files 
`` git add . `` or `` git add * `` 

### To Take snapshot of the added files thus git can track their version 
`` git commit -m " initial commit `` 
> '-m' is used to provide a message

###  to push the commits on the remote repo
`` git push origin main `` 
> use name of remote repo if name of your remote repo is not origin 


### Git command to add only modified files to add , commit and push
`` git add -u ``
